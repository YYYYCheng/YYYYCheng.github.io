algo-stl基本使用

2019.03.25 Yc learning  热度 ℃

stl

vector

  • 语法: vector<类型> 变量名;
  • 构造函数：vector v(5,1); // 生成5个1
  • 二维动态数组: vector<vector > vec2(n, vector(m,0));

set

  • 插入insert();
  • erase() 删除一个元素
  • count() 统计集合中某个元素的个数
  • size() 获取元素个数
  • clear() 清空
  • 迭代器 set::iterator it=(set类型)变量.begin();
  • 内部有序，不能有重复元素。

map

  • map<类型1， 类型2> v;
  • 向映射中插入新映射需要pair。pair<string, int> p;
  • make_pair(v1, v2);返回由v1、v2初始化的pair
  • 如插入key前已经存在，不会插入新的value代替原来的value。

queue

  • queue q;//定义队列
  • q.push(1);//入队
  • q.pop();//出队
  • q.front();//返回队首元素
  • q.empty();判断是否为空
